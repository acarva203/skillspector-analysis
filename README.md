# skillspector-analysis

This is a [Next.js](https://nextjs.org) project bootstrapped with [v0](https://v0.app).

## Usage

```Is this AI agent skill safe to install?```

Paste a public GitHub repository and SkillSpector scans it for prompt injection, data exfiltration, dangerous code, supply-chain risks and more — then returns a 0–100 risk score with a clear recommendation.
